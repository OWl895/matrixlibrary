# Решение задач вычислительной математики с помощью обыкновенных дробей

Данная программа реализует базовые арифметические операции над обыкновенными дробями с использованием библиотеки **GNU MP (GMP)**. Это позволяет работать с числами практически неограниченной разрядности без потери точности.

## Особенности реализации

*   **Структура данных**: Дробь представлена структурой `Fraction`, содержащей числитель и знаменатель типа `mpz_t` (целые числа GMP).
*   **Автоматическое сокращение**: После каждой операции вызывается функция `frac_reduce`, которая находит НОД (наибольший общий делитель) и сокращает дробь.


## Реализованные функции

*   `frac_add` — сложение двух дробей.
*   `frac_sub` — вычитание.
*   `frac_mul` — умножение.
*   `frac_div` — деление (с проверкой на ноль).
*   `frac_print_decimal` — вывод значения дроби в десятичном виде с заданной точностью.

## Компиляция и запуск

Для сборки необходим компилятор `gcc` и установленная библиотека `libgmp-dev`.
