# Лабораторная работа: Высокоточные вычисления (GMP & MPFR)

## Описание задачи
Цель работы — продемонстрировать критическую разницу между стандартными типами данных (`double`) и библиотеками произвольной точности при работе с плохо обусловленными выражениями.

Вычисляется значение функции:
$$f(x, y) = \frac{1}{y^6 - 3xy^5 + 5x^3y^3 - 3x^5y - x^6}$$

Где $x$ и $y$ — это **46-е** и **47-е** числа Фибоначчи соответственно.

### Математическая особенность
При подстановке последовательных чисел Фибоначчи знаменатель этой функции теоретически всегда равен **1**. Однако из-за возведения больших чисел в 6-ю степень промежуточные значения становятся огромными, что приводит к потерям точности при использовании стандартного 64-битного `double`.

## Требования
Для компиляции необходимы:
*   Библиотека **GMP**
*   Библиотека **MPFR**
*   Компилятор **GCC**

gcc main.c -o fib_calc -lgmp -lmpfr -lm
