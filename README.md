# Машинное представление вещественных чисел

## Описание

Данная программа автоматически создает наборы случайных чисел и показывает их внутреннее машинное представление (знак, экспонента, мантисса) для различных форматов точности.

### Основные возможности

- **Генерация случайных чисел** в заданном диапазоне с равномерным распределением
- **Поддержка всех основных форматов** IEEE 754:
  - 32 бита (float) — одинарная точность
  - 64 бита (double) — двойная точность
  - 128 бит (long double) — расширенная точность
- **Настраиваемая точность** округления (количество знаков после запятой)
- **Создание индивидуальных вариантов** для целой группы студентов
- **Автоматический расчет погрешности** при обратном преобразовании

## Алгоритм

1. Программа читает параметры из файла `input.txt`
2. Генерирует указанное количество случайных чисел в заданном диапазоне
3. Для каждого числа выполняет преобразование в машинное представление
4. Выполняет обратное преобразование для оценки точности

## Входные данные (input.txt)

Файл `input.txt` должен содержать пять параметров, разделенных пробелами или переносами строк:
1) Количество вариантов;
2) Количество заданий;
3) Разрядность;
4) Начало и конец диапазона;
5) Точность;

## Выходные данные
Каталоги `Задания` и `Проверка` с указанным количеством вариантов и исходных чисел.
