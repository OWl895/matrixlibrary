# Тест МИллера-Рабина

## Описание работы
Реализация вероятностного теста простоты Миллера-Рабина для проверки больших чисел на простоту. Тест особенно эффективен для чисел длиной сотни и тысячи разрядов.

## Теоретическая основа

### Алгоритм Миллера-Рабина
Пусть n — проверяемое число, и n - 1 = 2^s · d, где d — нечётно. Тогда для простого n и любого a из Z_n выполняется хотя бы одно из условий:

1. a^d ≡ 1 (mod n)
2. Существует целое число r < s, такое что a^(2^r · d) ≡ -1 (mod n)

Если ни одно из условий не выполняется — число составное.

### Вероятность ошибки
При k итерациях вероятность ошибки составляет 2^(-2k):
- k = 5 → ~10^(-3) (быстрая проверка)
- k = 10 → ~10^(-6) (стандартная надёжность)
- k = 20 → ~10^(-12) (высокая надёжность)
- k = 40 → ~10^(-24) (максимальная надёжность)


## Требования

### Для компиляции C программы
- GCC
- Библиотека GMP (GNU Multiple Precision Arithmetic Library)

Установка GMP:
```bash
sudo apt-get update
sudo apt-get install libgmp-dev
```
