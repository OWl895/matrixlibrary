
# Дроби и матричные операции (GMP)

C++ библиотека для высокоточных вычислений с матрицами. Использование библиотеки **GNU MP (GMP)** позволяет работать с рациональными числами (дробями) любой сложности без потери точности, характерной для типов `double` или `float`.

## Реализованные алгоритмы

В библиотеке реализованы 8 ключевых методов линейной алгебры:

1.  **Умножение матрицы на вектор**
2.  **Классическое умножение матриц**
3.  **Блочное умножение матриц**
4.  **Алгоритм Штрассена**
5.  **Метод Гаусса**
6.  **Метод Жордана-Гаусса**
7.  **Инверсия методом Гаусса**
8.  **Инверсия методом Жордана-Гаусса**

## Технологии и зависимости

*   **Язык**: C++17
*   **Библиотека**: [GNU MP (GMP)](https://gmplib.org) — для работы с числами произвольной точности.
*   **Тип данных**: `mpq_t` (рациональные числа с автоматическим сокращением дробей).
